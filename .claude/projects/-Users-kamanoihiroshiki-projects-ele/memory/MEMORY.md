# ele — Project Memory

## What ele is
CLI-first pseudo-RAW preprocessing engine. Converts JPEG/PNG → 16-bit TIFF or Linear DNG.
ele is NOT an editor. Outputs prioritise editability for Lightroom / Photoshop / Capture One.
